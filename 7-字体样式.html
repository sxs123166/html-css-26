1.字体样式 字体样式 说明 color 设置字体颜色 font-family 字体族 font-size 字体大小 font-style 字体风格 font-weight 字体粗体 text-decoration 字体装饰 1.1 color 1)关键字 red/green等 2)十六进制 开发最常用 p{ color: #f00; } 3)rgb格式，rgb()函数接受三个参数，分别表示颜色的红、绿和蓝。也是设计稿复制的，例如p { color: rgb(255,0,0); } rgba(255,0,0, 0.3)文字透明 0完全透明 1 完全不透明 1.2 字体族font-family 给定一个有先后顺序的，由字体名或者字体族名组成的列表来为选定的元素设置字体 body { font-family: "Arial", "宋体", sans-serif; } 属性值用逗号隔开。浏览器会选择列表中第一个该计算机上有安装的字体。 实际开发中，都是规定好的直接复制即可，无需记忆。 1.3字体大小font-size font-size属性用于设置字体的大小。像素px CSS 像素（CSS Pixel）是 CSS 中用于定义长度、尺寸的单位（简写为px） 绝对单位 不同浏览器默认字体大小不一样，谷歌默认 16px，建议给body标签统一指定大小，做到浏览器统一 body { font-size: 16px; } 1.4 字体风格font-style font-style属性用于设置字体的风格。 font-style: normal; 正常字体 font-style: italic; 斜体字体 最常见：让em或者i标签取消默认倾斜 em { font-style: normal; } 1.5 字体粗体 font-weight font-weight属性用于设置字体的粗体。 font-weight: normal; 正常字体 font-weight: bold; 粗体字体 数字属性值：取值 100～900 之间，常用就是 400 和 700 400:正常粗细 700:加粗 最常见：让strong或者b标签取消默认粗体 strong { font-weight: normal; } 1.6 字体装饰text-decoration 设置/取消字体上的文本装饰 text-decoration属性用于设置字体的装饰。 text-decoration: none; 无装饰 text-decoration: underline; 下划线 text-decoration: line-through; 删除线 text-decoration: overline; 上划线 最常见：让a标签取消默认下划线 a { text-decoration: none; } 2.文本布局 字体样式 说明 text-align 文本对齐 text-indent 首行缩进 letter-spacing 文本字符间距 line-height 行高 2.1 文本对齐 text-align 控制文本在它所在的块级盒子内如何水平对齐 使用场景： 1）文本/图片在盒子水平对齐 2）文章文字两端对齐 属性值： left 左对齐 center 水平居中对齐 right 右对齐 justify 自动改变字间距，两端对齐 2.2 首行缩进 text-indent 设置块级元素中文本行前面空格（缩进）的长度,对行内元素无效 使用场景： 1）段落首行缩进2 个字的效果 2）logo隐藏文字效果 属性值： 数字 常见单位是 em,相对单位，本元素的文字大小 1em等于当前元素的字体大小，如果当前元素没有大小，则按照父元素文字大小 p { text-indent: 2em; } 2.3 字间距 letter-spacing 设置文本字符之间的间距 使用场景： 1)调整字与字之前距离，用户体验更好 属性值：数字 p { letter-spacing: 2px; } 2.4 行高 line-height 设置文本每行之间的高 使用场景： 1）设置多行文本之间的上下间距 2）让单行文本居中 属性值： 1）数字px 2）数字不带单位（当前字体大小的倍数） p { line-height: 1.5; } 行高为盒子高度，可以设置文本单行居中。 p { line-height: 30px; } 行高包含上间距和下间距 单行文本垂直居中的原理为改变文字的上下间距，使其居中 3.font简写 font简写属性在一个生命中设置多个字体属性 使用场景： 1.给整个页面设置字体相关样式 语法： font: font-style font-weight font-size/line-height font-family; 其中 font-size和 font-family是必须写 其他可以省略，默认显示 属性有严格的书写顺序 body { font: 14px/1.5 "宋体"; } 4.CSS 继承性 子元素自动继承父元素的某些 CSS 样式属性 主要集成跟文字相关的样式属性，比如字体、颜色、文本对齐等 但是子元素定义自己样式，会优先自己样式
